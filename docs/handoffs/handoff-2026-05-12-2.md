## Key Decisions and Why

### Icon-left over watermark for NPCModal speaker visual
- **What:** Full-opacity sprite in a fixed-width left column. NOT a low-opacity right-aligned watermark.
- **Why:** Both variants were sandboxed via WIP commit + swap. Icon-left won on legibility at placeholder-sprite scale — watermark felt softer but harder to parse. Doc notes the rejected option in §23 so if real illustration art ever lands, the call could be revisited.
- **Alternative:** Watermark right-aligned at ~18% opacity, text flowing over.

### Inline icons in DecisionModal (over absolute positioning)
- **What:** Icon rendered inline alongside the prompt (options phase) and chosen-option label (flavor phase) via flex-row `[ICON] [text]`. NOT `position: absolute, top-{N}, {side}-{N}`.
- **Why:** Two prior absolute-positioned iterations (top-right, then top-left) hid the "YOU CHOSE" header in flavor phase because the icon overlapped that line. User intervention with a sketch of the desired layout solved it.
- **Alternative:** Various absolute positions — all hid content. **Decoration overlays hide load-bearing labels — don't do it.**

### Rewind door = forward door color
- **What:** Rewind door fill `palette.surface` → `palette.accent`; stroke `palette.inkMuted` → `palette.ink`; width 1.5 → 2; opacity 0.85 → 1. Now visually identical to the forward door.
- **Why:** User playtest feedback: subdued palette was making the rewind door blend with desks and other room furniture. Position alone (bottom-left vs right-wall-middle) differentiates the two doors — color doesn't need to.
- **Alternative:** Keep subdued but reduce stroke; ink-only fill. Position-only differentiation won.

### Month 1 unreachable in backward replay
- **What:** `previousReplayableMonth()` floors at 2; `enterReplay` reducer refuses target < 2 defensively.
- **Why:** The 2020 NarrativeRoom intro is a one-time framing beat. Walking back to it breaks the spell.
- **Alternative:** Reachable like other months (rejected per user); skip narrative rooms generally (rejected — year-transition narratives ARE replayable, just not the opener).

### `/punch` skill goes user-global, not project-scoped
- **What:** Source of truth at `~/.claude/skills/punch/SKILL.md`, removed from repo. New Step 0 asks before creating `docs/logs/time-log.md` in a fresh project.
- **Why:** Project-level `.claude/skills/` is not auto-discovered in current Claude Code — the skill wasn't appearing in the slash-command picker. User-global IS discoverable (validated via working `/punch` invocation at end of sprint). Trade-off: skill is now cross-project, not tied to this repo. Step 0 makes the cross-project case safe.
- **Alternative:** Try to register a project-skills path via `update-config` or settings.json (didn't pursue — unclear if supported); use `anthropic-skills:skill-creator` (didn't need — direct file move worked).

### Month-delta-float animation: drop the `-50%` translate
- **What:** Remove `transform: translate(-50%, ...)` from the `month-delta-float` keyframe; emits now anchor right-edge-consistent via parent `right: 0`.
- **Why:** Three prior attempts to fix "6+ months emit is too big" by reducing font size didn't satisfy. Actual cause was in the CSS keyframe: `-50%` shifts by 50% of element width, so wider strings (`+6 months pass`) drifted further left and looked bigger. **When the obvious fix doesn't satisfy, the source is one layer up.**
- **Alternative:** More font-size reductions; shorter wording; tier-specific positioning logic — all over-engineering relative to the actual cause.

## Dead Ends

- **Absolute-positioned DecisionModal icons (top-right, then top-left).** Both hid the `YOU CHOSE` header in flavor phase. Two iterations before settling on inline flex-row. **Lesson: decoration overlays hide load-bearing labels; build icons into the content flow, not on top of it.**

- **Watermark variant of NPCModal speaker visual.** Built and committed as a WIP checkpoint, then thrown away after the user compared with icon-left. **Lesson: at small placeholder-sprite scale, watermark aesthetic doesn't read — needs real illustration art before it's worth revisiting.**

- **Project-level `.claude/skills/` path for the `/punch` skill.** Initial implementation committed the SKILL.md to the repo with a `.gitignore` exception. Slash-command picker never showed it. **Lesson: in current Claude Code, user-global (`~/.claude/skills/`) is the auto-discovery path; project-level isn't.**

- **Three font-size reductions on the 6+ month emit.** Tier sized down from 24px → 19px → 18px → 18px (identical to 2-5 tier). None of them fully addressed the "still too big" complaint because the actual cause was the keyframe's `-50%` translate scaling with width. **Lesson: when an obvious-site fix doesn't satisfy, look one layer up — animation transforms, parent flex behavior, or CSS specificity often hide the real cause.**

- **Door label positions overflowing canvas.** First label placement attempt put `↩ return to {liveMonth}` at `x = door.x + door.width/2, textAnchor="middle"` — text extended past the canvas right edge and clipped. Fixed by switching to `textAnchor="end"` anchored at `ROOM_VIEWBOX.width - 10`. **Lesson: SVG text anchoring matters; middle-anchor at edge means half the text is off-screen.**
